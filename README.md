# function-microservice
Applicación Dymmu que devuelve la información de una Empresa.

### Tech

Function-microservice usa los siguientes proyectos:

* Spring Cloud Function

### Instalación

Function-microservice requiere de Java8+.

```sh
$ ./gradlew shadowJar
```
